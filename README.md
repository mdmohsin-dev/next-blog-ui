# Blog Platform

A feature-rich blog platform that allows users to explore, read, and manage blog content through a clean and responsive interface. The application includes secure authentication, dynamic blog management, seamless user experience, and efficient data handling to deliver a modern content publishing solution.

## ✨ Features

* Secure authentication with NextAuth
* Create and publish blog posts
* Browse latest blogs on the homepage
* Read detailed blog content
* Responsive design for all devices
* PostgreSQL database integration
* Prisma ORM for database management
* Type-safe development with TypeScript

## 🛠️ Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS
* ShadCN UI
* NextAuth

### Backend

* Prisma ORM
* PostgreSQL
